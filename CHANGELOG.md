# Changelog

## [Unreleased]

### Added

- Desktop GUI for exploring historical marine weather data.
- Interactive earth map with geographic selection and hover coordinates.
- Weather summary collection views, including main and wind rose screens.
- Support for importing local ICOADS files or downloading ICOADS files from NOAA by year range.
- CLI commands for fetching ICOADS IMMA files, processing them into parquet, and generating lattice files.
- Save and open support for weather summary collections.
